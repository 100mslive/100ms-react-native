# react-native-hmssdk

React native wrapper for 100ms SDK

## Installation

```sh
npm install react-native-hmssdk
```

## Usage

```js
import Hmssdk from "react-native-hmssdk";

// ...

const result = await Hmssdk.multiply(3, 7);
```
